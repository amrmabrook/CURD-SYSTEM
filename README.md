# CRUD System

A simple CRUD (Create, Read, Update, Delete) application built with **HTML, CSS, and Vanilla JavaScript**.

## Features
- Add new items
- Edit existing items
- Delete items
- Data persistence using localStorage

## How to Run
- Clone the repository
- Open `index.html` in your browser
